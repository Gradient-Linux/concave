# Flow Suite

## Purpose

Flow is the orchestration and observability suite for experiment tracking, pipelines, serving, storage, and dashboards.

## Services

- `gradient-flow-mlflow`
- `gradient-flow-airflow`
- `gradient-flow-prometheus`
- `gradient-flow-grafana`
- `gradient-flow-store`
- `gradient-flow-serve`

## Ports

- `5000` MLflow
- `8080` Airflow
- `9090` Prometheus
- `3000` Grafana
- `9001` MinIO console
- `3100` BentoML

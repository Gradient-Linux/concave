# Suite Guide

## Boosting

CPU-first classical machine learning environment with a long-running Python container, JupyterLab, and MLflow.

## Neural

GPU-oriented training and inference stack built around PyTorch, CUDA runtime images, and JupyterLab.

## Flow

Multi-service MLOps suite covering MLflow, Airflow, Prometheus, Grafana, MinIO, and BentoML.

## Forge

Custom suite assembly mode that lets the user select components from the other suites and generates a tailored Compose file.
